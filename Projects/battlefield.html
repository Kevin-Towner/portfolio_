Favoriting a Server

Every game has it's weird quirks..


Battlefield 4 is the 13th game installment to the Battlefield Franchise making, it a 6-year old game; originally for last generation consoles (Playstation 3, Xbox 360 and PC). For a 6-year old game, Battlefield holds up extremely well and is probably my favorite online, FPS, to date.


Every game has it's weird quirks..


For me, I always felt that Battlefield 4 made it inconvenient and slightly awkward to save a favorite online server. For most online console games, as well as web and mobile apps, the ability to 'favorite' an object is something many users take for granted. With a handheld device, there is usually an icon or a bookmark tab for the user to press, and for an online game the ability will usually be somewhere within the start menu or on the server row

There's only one location to "favorite" a server
