Favoriting a Server

Could be more accessible

Despite it's initial release in 2013, Battlefield 4 is one of my favorite video games of all time. I always felt Battlefield 4's "add server to favorites" option was not very accessible to it's online players. For most online console games, as well as web and mobile apps, the ability to 'favorite' an object is accessible. With a handheld device, there is usually an icon or a bookmark tab for the user to press, and for an online game the ability will usually be somewhere within the start menu or on the server row

Assumptions

  • Users who are unfamiliar with Battlefield 4 will find the process of favoriting a server to be unintuitive.

  • In the middle of online gameplay, the average user will first look to favorite the server from the start menu

  • Once the user navigates to the "server info" screen, he or she will take longer than 2 seconds to discover the button-press for "favorite"

There's only one location to "favorite" a server

Observation

I conducted two observational research studies on experienced gamers. The two participants are adept at console gaming and have experience playing first-person shooters like: Call of Duty and Overwatch. I took a naturallistic approach since the need for a labratory was not necessary, and I didn't want the participants' natural inclinations to be sabotaged by a controlled setting.

User 1 Scenerio

  • Gameplay commensed an hour and a half prior to dinner reservations

  • The observational study began mid match (Conquest Large)

  • I ask the participant if he would mind "saving" the server so we could go back to it later

User 2 Scenerio

  • There was no time constraints

  • The observational study began from Battlefield 4's home screen

  • I asked the participant to find a good "Operation Locker", hardcore, match and add it to my favorites (Why I particularly chose Operation Locker is irrelevant) while I work on my computer
