Favoriting a Server

Could be more accessible

For me, I always felt that Battlefield 4 made it inconvenient and slightly awkward to save a favorite online server. For most online console games, as well as web and mobile apps, the ability to 'favorite' an object is accessible. With a handheld device, there is usually an icon or a bookmark tab for the user to press, and for an online game the ability will usually be somewhere within the start menu or on the server row


I will assume...

  • Users who are unfamiliar with Battlefield 4 will find the process of favoriting a server to be unintuitive.

  • In the middle of online gameplay, the average user will first look to favorite the server from the start menu

  • Once the user navigates to the "server info" screen, he or she will take longer than 2 seconds to discover the button-press for "favorite"

There's only one location to "favorite" a server

Current Status

My redesign of Battlefield 4's, 'Add to Favoritres', functionality is ongoing. In the coming week I will be sharing my observational research results.
